**What to do to get this script running**

##

1. Download [Python](https://www.python.org/downloads/).
2. Install it.
3. Open Command Prompt and type **pip install pytube**
4. **Download** the [YouTube_Downloader.exe](https://github.com/IdealJakob/YouTube-Downloader/raw/main/YouTube_Downloader.exe) or create a python script file  and copy paste the code from [YouTube_Downloader.py](https://github.com/IdealJakob/YouTube-Downloader/blob/5169b21fdf4497bfb627d74c119fe82bc57096f3/YouTube_Downloader.py) into this file.
5. Run the .exe or .py and use the program :)
